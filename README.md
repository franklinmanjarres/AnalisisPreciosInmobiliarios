##  Descripción
Proyecto de Machine Learning para predecir precios de propiedades 
inmobiliarias. Se compararon modelos de regresión y se seleccionaron 
las variables más relevantes para mejorar la precisión.

## Objetivo
Identificar el mejor modelo de regresión y las variables clave que 
influyen en el precio de una propiedad.

##  Resultados principales
- Modelo seleccionado: **HuberRegressor** (robusto frente a valores atípicos)
- MAE reducido de **182,847 → 169,465** (mejora del 7.3%)
- Variables más influyentes: tipo de propiedad, número de habitaciones, 
  ubicación geográfica

##  Tecnologías utilizadas
- Python 3
- Pandas · NumPy
- Scikit-learn (LinearRegression, HuberRegressor, StandardScaler)
- Matplotlib · Seaborn
